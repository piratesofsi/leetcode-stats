# LeetCode Stats

A clean and interactive web application to visualize LeetCode progress in real time.  
Enter a LeetCode username to instantly view problem-solving statistics with a modern UI.

Live Demo: https://leetcode-stats-ten.vercel.app

---

## Features

- Search any LeetCode username
- Visual progress indicators for Easy, Medium, and Hard problems
- Key statistics overview:
  - Total problems solved
  - Total submissions
  - Submission success rate
  - Global ranking
- Keyboard support (press Enter to search)
- Refresh option to reset and search again
- Smooth UI transitions between compact and expanded views
- Real-time data fetching

---

## Tech Stack

- HTML5
- CSS3 (Flexbox, gradients, transitions)
- Vanilla JavaScript
- LeetCode Stats API
- Deployed on Vercel

## How It Works

1. The user enters a LeetCode username.
2. The application fetches real-time data from the API.
3. The fetched data is processed and rendered dynamically.
4. The interface expands smoothly to display detailed statistics.

---

## Notes

- Submission success rate is calculated using accepted submissions versus total attempts.
- Some values may differ slightly from the official LeetCode profile due to API limitations.
